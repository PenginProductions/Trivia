This Python project asks a series of Trivia questions in a fun way. 
____________________________________________________________________________________________________________________________________________________________________
To Run:

	1) Launch _trivia.bat
____________________________________________________________________________________________________________________________________________________________________
Tips:

All of the round titles, questions, choices, and answers are stored in the trivia.txt file and is read by the program;
if you would like to add more questions just add them onto the end of the file in the following order:
	(one line each)
	-Round title
	-Question
	-Choice 1
	-Choice 2
	-Choice 3
	-Choice 4
	-Answer (as an integer)
	-Answer explaination

If you would like to change the first question, start your entry after the line that says "An Episode You Can't Refuse".
